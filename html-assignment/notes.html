5 HTML Tags

<head>
This element is a child tag of <html>. We fill metadata inside which will not be displayed but for the use of the browsers.
<meta charset="UTF-8">& <title> are required and always be used.
<article>
The element is used to contain independent piece of content. An article should make sense standing alone.
<strong>
This tag emphasizes text which is highly important. The content inside will be displayed in bold, and screen readers will change their voices for the text.
<a>
We use the anchor tag to link from one page to another. The attribute <href> providing the link's destination should be used together.
<form>
The tag creates an HTML form for input. It contains two attributes: <action> & <method> which specify where to send and how to send separately.
